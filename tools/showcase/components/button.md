Use button to handle click events.

```py
q.page['form'] = ui.form_card(box='1 1 1 1', items=[
    ui.button(name='button', label='Button')
])
```

## Primary

Use primary to I dunno.

```py
q.page['form'] = ui.form_card(box='1 1 1 1', items=[
    ui.button(name='button', label='Button', primary=True)
])
```
