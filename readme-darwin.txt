TOUR

    Step 1: Start the Q development server:
            $ ./qd

    Step 2: Run the tour:
            $ python3 -m venv venv
            $ ./venv/bin/pip install -r examples/requirements.txt
            $ ./venv/bin/python examples/tour.py

    Step 3: Go to http://localhost:55555/tour
